Apply consistent, professional styling to presentations, documents, reports, and HTML pages with curated themes.

## Available Themes

The following 10 professionally designed themes are available:

1. **Ocean Depths** - Professional and calming maritime theme
2. **Sunset Boulevard** - Warm and vibrant sunset colors  
3. **Forest Canopy** - Natural and grounded earth tones
4. **Modern Minimalist** - Clean and contemporary grayscale
5. **Golden Hour** - Rich and warm autumnal palette
6. **Arctic Frost** - Cool and crisp winter-inspired theme
7. **Desert Rose** - Soft and sophisticated dusty tones
8. **Tech Innovation** - Bold and modern tech aesthetic
9. **Botanical Garden** - Fresh and organic garden colors
10. **Midnight Galaxy** - Dramatic and cosmic deep tones

## Application Process

### Step 1: Show Theme Showcase
Display the `theme-showcase.pdf` file to allow users to see all available themes visually.

### Step 2: Ask for Theme Selection
Ask which theme to apply to the artifact.

### Step 3: Wait for Confirmation
Get explicit confirmation about the chosen theme.

### Step 4: Apply the Theme
Apply the selected theme's colors and fonts consistently throughout the artifact.

## Theme Structure

Each theme includes:
- **Cohesive color palette** with hex codes
- **Complementary font pairings** for headers and body text  
- **Distinct visual identity** suitable for different contexts and audiences

## Usage Guidelines

### Theme Selection
- Consider the artifact's purpose and audience
- Match theme personality to content tone
- Ensure proper contrast and readability
- Maintain visual consistency across all elements

### Application Standards
- Apply colors consistently throughout
- Use specified fonts for headers and body text
- Ensure proper contrast ratios
- Maintain the theme's visual identity

## Custom Theme Creation

When existing themes don't fit the artifact:

### Generate Custom Theme
Based on provided inputs:
1. Create a new theme similar to existing ones
2. Choose appropriate colors and fonts based on description
3. Give the theme a descriptive name
4. Apply the same structure as existing themes

### Theme Specification Format
```markdown
# Theme Name

## Color Palette
- Primary: #hexcode
- Secondary: #hexcode
- Accent: #hexcode
- Background: #hexcode
- Text: #hexcode

## Typography
- Headers: Font Name
- Body: Font Name

## Usage Guidelines
- When to use this theme
- Best practices for application
```
