## Skills by Category

### Document Processing (Standard/Pro)

These skills enable creation, editing, and extraction from common document formats.

---

#### PDF

**Purpose:** Comprehensive PDF manipulation toolkit

**What It Enables:**
- Extract text and tables from PDFs
- Merge and split PDF documents
- Fill PDF forms
- Create new PDFs
- Add watermarks
- Extract images
- Apply password protection
- OCR for scanned PDFs

**Example Use Cases:**
- Extract tables from financial reports
- Merge multiple documents into one
- Fill form-based documents automatically
- Create reports from scratch
- Process and organize PDFs at scale

**Example Usage:**

```
@Master: Extract all tables from this PDF and convert to Excel

@Full-Stack Developer: Create a PDF report with the data and styling

Agent with PDF skill: Fill out this PDF form with the following information: [details]

@Master: Merge these 5 PDFs into a single document
```

**Key Capabilities:**
- Text extraction with layout preservation
- Table extraction and conversion
- PDF creation from scratch
- Merging and splitting
- Form filling
- Watermarking
- Image extraction
- OCR for scanned documents
- Password protection
- Metadata manipulation

**Integration:**
Works with all agents. Full-Stack Developer frequently uses for document generation.

## Skill Discovery Tips

### Need to Extract Data?
Use **PDF** or **XLSX** to extract tables and information from documents.

### Creating Documents?
- Formal: **DOCX** or **PDF**
- Data-heavy: **XLSX**
- Presentations: **PPTX**
- Quick export: **PDF**

### Designing Visuals?
- Original art: **Canvas Design**
- Theming: **Theme Factory**
- Brand consistency: **Brand Guidelines**
- Interactive: **Artifacts Builder**
- Algorithmic: **Algorithmic Art**
- Animated: **Slack GIF Creator**

### Building Systems?
- Internal communications: **Internal Communications**
- Custom tools: **MCP Builder** + **Artifacts Builder**
- Extend toolkit: **Skill Creator**
- Test coverage: **Web App Testing**

### Common Workflows

#### Document Creation Workflow
1. Create content with agent
2. Export to **DOCX** or **PDF**
3. Apply **Brand Guidelines**
4. Add **Theme**

#### Design Workflow
1. Create wireframe with **UX Expert**
2. Apply visual design with **Canvas Design**
3. Apply theme with **Theme Factory**
4. Build interactive prototype with **Artifacts Builder**

#### Data Analysis Workflow
1. Extract data with **PDF** or **XLSX**
2. Analyze with agent
3. Create visualization with **XLSX** or **Algorithmic Art**
4. Present with **PPTX**

#### Integration Workflow
1. Design tool interface with agent
2. Build MCP server with **MCP Builder**
3. Create UI with **Artifacts Builder**
4. Test with **Web App Testing**
